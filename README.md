# AWS Cloud-Based CRM System 🚀

A full-stack, scalable CRM solution built with **Next.js** and **AWS** services.

## ✨ Key Features
* **User Authentication:** Secure login/signup via AWS Cognito (RBAC).
* **Serverless Backend:** High-performance API using AWS Amplify and Lambda.
* **NoSQL Database:** Managed data using DynamoDB for fast queries.
* **Dashboard:** Real-time analytics and task management UI.

## 🛠 Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Vercel
- **Backend:** Node.js, AWS Amplify
- **Database:** Amazon DynamoDB
- **Auth:** AWS Cognito

## 🚀 How to Run Locally
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Set up AWS credentials.
4. Run: `npm run dev`
